# Projeto Banco de Dados – Oficina Mecânica

Este projeto tem como objetivo a modelagem e implementação de um banco de dados relacional para o contexto de uma oficina mecânica.

## Contexto
O sistema foi projetado para controlar clientes, veículos, mecânicos, ordens de serviço e os serviços executados em cada ordem.

## Modelo Lógico
O modelo relacional contempla as seguintes entidades:
- Cliente
- Veículo
- Mecânico
- Ordem de Serviço
- Serviço
- Item de Ordem de Serviço

Os relacionamentos foram implementados por meio de chaves estrangeiras, garantindo integridade referencial.

## Implementação
O banco foi criado utilizando SQL padrão, com scripts para:
- Criação das tabelas
- Inserção de dados para testes
- Execução de consultas SQL com SELECT, WHERE, ORDER BY, HAVING, JOIN e atributos derivados

## Consultas
As consultas desenvolvidas permitem:
- Visualização de clientes e veículos
- Acompanhamento de ordens de serviço
- Cálculo de valores totais
- Análise de serviços executados
